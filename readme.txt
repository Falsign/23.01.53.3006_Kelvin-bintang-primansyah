# 3006 kelvin bintang primansyah

UTS - Sistem Penjualan Berbasis Web

## Deskripsi
Aplikasi ini merupakan sistem penjualan sederhana yang menggunakan database MySQL dengan 10 tabel.

## Backend
Backend dibuat menggunakan PHP dan menyediakan layanan API berbasis JSON.

Endpoint API:
- /api/produk.php
- /api/pelanggan.php

## Frontend
Frontend dibuat menggunakan HTML dan JavaScript untuk mengkonsumsi API dan menampilkan data produk.
